This project is made by Harold Kwan.

You can click the train button to train a neural network to classify different colors. The training process can take up to a few minutes.

The neural network may not perform optimally due to the lack of data in our database. You can help expanding it by putting in data here.

The neural network is built with TensorFlow.js. Learn more about Tensorflow here.

This project is inspired by the color classifier made by The Coding Train (Daniel Shiffman). You can check out his youtube channel here.
